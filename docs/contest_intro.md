# MyFRP - 基于 asyncio 的轻量级内网穿透工具

## 项目简介

MyFRP 是一款基于 Python asyncio 从零实现的轻量级反向代理（内网穿透）工具，支持 TCP 和 UDP 双协议端口转发。项目参考 FRP 设计理念，以纯 Python 异步编程为核心，实现了控制通道与数据通道复用、心跳保活、TLS 加密、访问控制、流量统计等完整功能，并提供了 React + FastAPI 架构的可视化 Web 管理面板。

## 技术亮点

### 1. 纯 asyncio 单线程异步架构
- 全程基于 Python asyncio 事件循环，单线程处理大量并发连接
- 控制消息与数据消息复用同一控制连接，减少连接开销
- 所有读写操作均以协程方式实现，通过 await 让出执行权，无多线程锁竞争
- 服务端使用 asyncio.Event 实现数据转发的同步等待机制

### 2. 自研消息协议与通信机制
- 自定义二进制帧协议：4 字节大端序长度头 + JSON 消息体
- 9 种消息类型覆盖完整生命周期：LOGIN / REGISTER / NEW_CONN / INIT_CONN / PING / PONG / DATA / CLOSE / ERROR
- 每个通信链路通过 UUID 格式的 conn_id 唯一标识
- 反向连接设计：服务端无法主动连接内网客户端，通过控制通道下发 NEW_CONN 指令，客户端主动建立本地连接

### 3. 生产级可靠性保障
- **Token 认证**：连接建立后立即进行 LOGIN 握手，未认证不允许注册代理
- **TLS/SSL 加密**：控制通道全程支持 TLS 加密传输
- **断线重连**：客户端采用指数退避算法自动重连（初始 1s → 最大 60s）
- **心跳保活**：30 秒 PING/PONG 心跳，检测连接存活状态
- **空闲清理**：服务端定期扫描并清理超时空闲连接，防止资源泄漏
- **优雅关闭**：捕获 SIGINT/SIGTERM 信号，有序关闭所有连接与协程

### 4. 完善的安全访问控制
- **端口白名单**：限制客户端可注册的 remote_port 范围（如 "8080,9000-9100"）
- **IP 黑/白名单**：基于 ipaddress 模块支持 CIDR 格式的访问过滤
- **连接数限制**：max_connections 限制最大并发控制连接数

### 5. Web 可视化管理面板
- **技术栈**：React 18 + Vite + Ant Design 5 + FastAPI + WebSocket
- **功能模块**：
  - 📊 仪表盘：服务状态、连接数、流量总览与趋势图
  - ⚙️ 配置管理：表单 / JSON 双模式在线编辑，实时校验
  - 🚀 服务管理：frps / frpc 启停、重启控制，进程状态可视化
  - 📝 日志监控：WebSocket 实时日志流，支持筛选、搜索、下载
  - 🔌 连接监控：活跃连接列表，流量明细，手动断开
  - 👥 用户管理：JWT 鉴权，bcrypt 密码哈希，多角色支持

## 创新点

1. **控制与数据复用设计**：控制消息和数据消息通过同一控制连接传输，相比传统 FRP 独立数据通道方案，减少了连接建立开销，更适合轻量级场景

2. **NEW_CONN + INIT_CONN 双向握手机制**：服务端发送 NEW_CONN 触发客户端建连，客户端建连完成后回发 INIT_CONN，服务端通过 asyncio.Event 等待就绪后再开始转发，确保链路时序正确性

3. **UDP 代理的连接抽象**：将无连接的 UDP 数据报通过 conn_id 抽象为有状态的"连接"，复用 TCP 的连接管理与统计框架

4. **零依赖核心实现**：核心代理功能仅依赖 Python 标准库（asyncio / json / struct / ssl），不安装任何第三方包即可运行

## 应用场景

- **内网服务暴露**：将本地开发的 Web 服务、SSH 服务、数据库等安全地暴露到公网
- **远程办公**：居家办公时访问公司内网的各类系统与设备
- **IoT 设备管理**：远程管理部署在局域网内的物联网设备
- **开发测试**：临时将本地服务暴露给外部人员联调测试
- **跨网络访问**：在不同网络环境间搭建安全的数据转发通道

## 快速体验

### 启动服务端
```bash
python frps.py -c config/frps.json
```

### 启动客户端
```bash
python frpc.py -c config/frpc.json
```

### 启动 Web 管理面板
```bash
python webadmin.py
# 浏览器访问 http://localhost:7500
# 默认账号 admin / admin123
```
